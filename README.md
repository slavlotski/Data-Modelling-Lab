# Data Modelling Lab

An educational repository for learning data modelling — from relational fundamentals to advanced warehouse architectures (Kimball, Inmon, Data Vault, Anchor Modelling).

## Features

- **Theory modules** — modules with rich Markdown content

## Curriculum

| Modules | Topic |
|---------|-------|
| 1–4 | Relational Foundations |
| 5–9 | Dimensional Modelling (Kimball) |
| 10–11 | Inmon (Enterprise DW) |
 12–13 | Data Vault 2.0 |
| 14 | Anchor Modelling |
